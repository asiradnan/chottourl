## Overview

ChottoURL is a modern, fast, and efficient URL shortener that transforms long URLs into short, manageable links. Built with React and Vite on the frontend and Django REST Framework on the backend, it provides a seamless experience for creating, managing, and tracking shortened URLs.

---

## Tech Stack

### Frontend
- **React**: Modern JavaScript library for building user interfaces
- **Vite**: Fast build tool and development server
- **JavaScript**: Core programming language
- **TailwindCSS**: Styling and responsive design
- **HTML5**: Structure and semantic markup

### Backend
- **Django REST Framework**: API backend
- **Database**: SQLite
- **Deployment**: Self-hosted on private VPS with Nginx
- **Version Control**: Git, GitHub

---

## Features

- **URL Shortening**: Convert long URLs into short, shareable links
- **Click Analytics**: Track clicks, referrers, and usage statistics
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile
- **Fast Redirects**: Lightning-fast URL redirection
- **Copy to Clipboard**: Easy one-click copying of shortened URLs
